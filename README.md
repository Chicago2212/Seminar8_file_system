## Файловая система
*Функции:*
1. Добавление данных
2. Удаление данных
3. Изменение данных
4. Компирование данных
5. Перемещение данных
6. Вывод данных

**ui.py** - отвечает за взаимодействие программы с пользователем

**main.py** - главный файл, который вызывает всю цепочку

**db** - папка, в которой хранится 2 файла.
